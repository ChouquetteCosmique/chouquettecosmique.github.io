An UGC Ecosystem to create games


The Sandbox is a platform open to everyone to create games using in-house engine and consists of 3 complementary software as follows:

  • VoxEdit: Create graphic elements and animate them.
  • Game Client: Play existing games, multiplayer or not.
  • Game Maker: Create games using simplified programming tools.

    • All these software are under development and each update brings new features to the live version such as the Game Rules system which is a visual scripting tool that allows to use and manipulate values such as numbers, text, time, and true/false values for gameplay design needs.


    My work mainly concerns the creation of games using Game Maker, Prototyping and providing feedback using the upcoming new engine features and concept proposal to IP owners such as Ubisoft and Kodansha for Captain Laserhawk and Attack on Titan Experiences.

    From Concept to Completion

    Unlike in many other video game projects, level design plays a major role in the game's overall design and remains actively involved throughout nearly the entire production process. The level design team shares ownership of the game's structure and player experience with the game producer. Making them one of the key points of contact for the rest of the team, working closely with concept artists, level artists, sound designers, and interfacing with other disciplines as needed to ensure design consistency and gameplay cohesion.

    Consequently, it's not unusual for level designers to contribute to several projects at once. Given their strategic value at different stages of development, they are often allocated across teams as needed. While this cross-project involvement maximizes their impact, it also requires careful workload management to maintain quality and coherence across all deliverables.

    Bringing the glide on Attack on Titan: Invasion



    This project was initially designed to work without glide, as the functionality was not present in the 3Cs. However, I felt that this element was very important to deliver an experience worthy of Attack on Titan.

    I carried out several tests using the scripting tools at my disposal to implement a satisfying glide functionality.
    After some fine-tuning, this feature was validated by Kodansha and integrated into the overall concept of the experience as we know it.
